Программа математического моделирования Доплеровского лага.
Симулируется движение беспилотного подводного аппарата, разбитое на три участка.
В процессе моделирования расчитываются реальные параметры БПА (скорость, координаты) и параметры, измеренные с помощью доплеровского лага.
При моделировании учитывается модель погрешности доплеровсокго лага (неправильная установка антенны, изменчивость скорости звука в воде 
в зависимости от температуры воды, масштабный коэффициент, дрейф нуля)
Расчитываются ошибки, строятся графики реальных параметров и параметров с погрешностями. 
В программе используются математические модели реальных физических явлений, влияющих на работу стройства.

- В программе используется две библиотеки numpy; matplotlib.
